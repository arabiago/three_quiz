🔧 Linux基本コマンド一覧

🔍 1. ファイル・ディレクトリ操作

コマンド説明
lsディレクトリの内容を表示（ls -l, ls -a）
cdディレクトリを移動（cd /home/naoto）
pwd現在のディレクトリを表示
mkdir新しいディレクトリを作成
rmdir空のディレクトリを削除
touch空のファイルを作成
cpファイル・ディレクトリをコピー
mvファイル・ディレクトリを移動（名前変更も）
rmファイル・ディレクトリを削除（rm -r, rm -f）
treeディレクトリ構造をツリー表示（要インストール）

📄 2. ファイル内容の確認・編集

コマンド説明
catファイル内容を表示
less / more長いファイルをページ単位で表示
head / tailファイルの先頭・末尾を表示（tail -fはリアルタイム監視）
nano / vi / vimテキストファイルを編集

👤 3. ユーザー・権限管理

コマンド説明
whoami現在のユーザーを表示
idUIDや所属グループを表示
chmodパーミッション（アクセス権）を変更
chown所有者を変更
suユーザーを切り替え
sudo管理者権限でコマンドを実行

📦 4. パッケージ管理

Debian/Ubuntu系（APT）

コマンド説明
apt updateパッケージ情報を更新
apt upgradeパッケージをアップデート
apt install パッケージ名パッケージをインストール
apt remove パッケージ名パッケージを削除

RHEL/CentOS系（dnf / yum）

コマンド説明
dnf update / yum updateシステム全体を更新
dnf install パッケージ名インストール
dnf remove パッケージ名削除

🌐 5. ネットワーク関連

コマンド説明
ip a / ifconfigIPアドレス確認
ping アドレスネットワーク接続テスト
curlURLからデータ取得
wgetファイルをダウンロード
netstat / ssポートや接続の確認

🧠 6. システム情報確認

コマンド説明
uname -aカーネルなどの情報表示
top / htopリアルタイムのプロセス情報（htopは見やすく要インストール）
ps aux現在のプロセス一覧
df -hディスク使用状況
free -hメモリ使用状況
uptime稼働時間の確認

📝 7. ログと履歴

コマンド説明
historyコマンド履歴表示
journalctlシステムログの閲覧（systemd）
dmesgカーネルメッセージの表示
last最近ログインしたユーザーを表示

🔁 8. その他便利系

コマンド説明
man コマンド名マニュアルを見る
aliasコマンドの別名設定
date日付と時刻を表示
calカレンダーを表示
clear画面をクリア
exitセッション終了（ログアウト）
